Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM, such as, get element by its "id" getElementById(), get elements by there "class name" getElementsByClassName(), get elements by their "tag name" getElementsByTagName(), get the first element that matches a selector querySelector(), get all the elements that matches a selector querySelectorAll() etc.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerText is used when we want to manipulate a particular tag's text. innerHTML us used when we want to create a html like structure. textContent is just like innerText but it also return hidden contents.

Question-3: What is event delegation in the DOM?

Instead of adding event listeners directly to each child element, the event listener add to the parent of those child elements.

Question-4: What is event bubbling in the DOM?

It is a way of how a event propagate in a DOM hierarchy. When a event triggered it doesn't just stay within the element rather it bubble up from the target element to its parents.

Question-5: How do you create, add, and remove elements using JavaScript?

To create elements use document document.createElement("tagname"), inside this the argument will be the tag name which we wanted to create. For adding the elements we can use appendChild() and its argument will be element which we created already although there are many more ways to add elements in JavaScript. Lastly, to remove a element we can use removeChild() and its argument will be the element which we want to remove.
